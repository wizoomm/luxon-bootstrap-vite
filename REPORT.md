# Отчет о проекте: Приложение для отображения времени

## Описание проекта

Веб-приложение для отображения текущего времени в реальном времени. Приложение построено с использованием Bootstrap 5, Luxon для работы с датой и временем, и Vite в качестве инструмента сборки.

## Технологический стек

- **Bootstrap 5.3.8** — CSS-фреймворк для стилизации
- **Luxon 3.7.2** — библиотека для работы с датой и временем
- **Vite 7.2.4** — инструмент сборки и разработки
- **Sass 1.77.0** — препроцессор CSS

## Интерфейс приложения

### Главная страница

![Главная страница](png/first.png)

На главной странице отображается красная кнопка "Показать время", которая открывает модальное окно с текущим временем.

### Модальное окно

![Модальное окно](png/second.png)

Модальное окно содержит:
- Заголовок с информацией об авторе: "Выполнила: Кручинина Дарья"
- Текущую дату и время в формате `DD.MM.YYYY HH:MM:SS`
- Кнопку "Закрыть" для закрытия модального окна

## Команды для работы с проектом

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

### Сборка проекта для продакшена

```bash
npm run build
```

Собранные файлы будут находиться в папке `dist/`

### Просмотр собранного проекта

```bash
npm run preview
```

## Функциональность

1. **Отображение времени**: При нажатии на кнопку "Показать время" открывается модальное окно с текущей датой и временем
2. **Обновление в реальном времени**: Время обновляется каждую секунду
3. **Локализация**: Используется русская локализация для формата даты и времени
4. **Адаптивный дизайн**: Приложение корректно отображается на различных устройствах благодаря Bootstrap
